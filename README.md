# 🎲 Number Guessing Game

A simple and interactive Command-Line Interface (CLI) number guessing game built entirely in Python. The game selects a random number between 1 and 6, and your mission is to guess it in the fewest attempts possible!

## ✨ Features
* **Smart Feedback:** The game guides you by telling you if your guess is "Very high" 🔼 or "Very low" 🔽.
* **Attempt Counter:** Tracks and displays exactly how many tries it took you to win.
* **Seamless Replayability:** Play multiple rounds in a row without needing to restart the script.
* **Lightweight & Fast:** Pure Python code with no external libraries or dependencies required.

## 🛠️ Prerequisites
To run this game, you only need to have **Python 3** installed on your machine.
